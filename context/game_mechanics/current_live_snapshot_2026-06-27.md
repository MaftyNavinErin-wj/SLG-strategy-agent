## Cross-Game Implication

The latest layer reinforces the main contrast:

1. 无尽冬日 current depth is mostly vertical account growth plus alliance-event execution.
2. 率土之滨 current depth is the mature baseline of seasonal map geopolitics and alliance command.
3. 三国志战略版 current depth is mostly seasonal board state plus horizontal team/buildcraft variation.
4. 三国谋定天下 current depth is the test of whether lower friction and formal roles can preserve 率土-like alliance strategy.
5. All four depend on community interpretation because late-game optimization is not fully legible from the client alone.
