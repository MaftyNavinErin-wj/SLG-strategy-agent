# Topic Index

Status: living index  
Last updated: 2026-07-15

## Purpose

This file is the library catalog for the SLG strategy workspace. Use it after `context/agent_brief.md` when a question may touch multiple topics or when you need to find the right note quickly.

Do not put full analysis here. Each entry should stay short: what the topic is, when to use it, and which files hold the detail.

## Workspace Architecture

1. `context/agent_brief.md`: front desk. First-read orientation, routing rules, stable conclusions, volatile claims, and answer protocol.
2. `context/topic_index.md`: library catalog. Lists topics and points to the right files.
3. `context/decision_log.md`: judgment record. Captures framework decisions, status, and rationale.
4. Topic notes: detailed synthesis for each question area.
5. `context/game_mechanics/`: per-game mechanism notes and dated live snapshots.
6. `workflows/`: reusable research and output processes.
7. `sources/`: source indexes and reliability logs, not raw confidential content.
