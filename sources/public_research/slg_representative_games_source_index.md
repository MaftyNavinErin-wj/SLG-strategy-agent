# Representative SLG Public Research Source Index

Access date: 2026-06-27
Latest refresh: 2026-06-27

This file records public sources used for first-pass mechanism understanding. It is an index and reliability log, not a raw-content archive.

## Research Rules

1. Use public, accessible materials only.
2. Prefer official pages, game wikis, long-form breakdowns, and player guides with concrete mechanics.
3. Treat Zhihu and Tieba as player-perspective and question-discovery sources when pages are partially inaccessible.
4. Do not copy full answers or posts. Store distilled mechanism notes and URLs.
5. Mark uncertain claims as requiring in-game or multi-source verification.
